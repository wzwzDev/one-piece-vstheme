## 🔄 How to Switch:

1. **Press `Ctrl + Shift + P`**
2. **Type "Preferences: Open Settings (JSON)"**
3. **Replace the CSS file path with your desired background**
4. **Save the file**
5. **Restart VS Code or reload window**
